|------|------|------|
| navigate | URL 이동 | ✅ |
| screenshot | 화면 캡처 | ✅ |
| click | 버튼/링크 클릭 | ✅ |
| fill | input/textarea 입력 | ✅ |
| select | 일반 드롭다운 선택 | ✅ |
| evaluate | JS 실행 | ⚠️ 최소 반환값만 |
| React 커스텀 UI | 크몽 같은 것 | ❌ 불가 |
| 파일 업로드 | setInputFiles | ❌ 없음 |
