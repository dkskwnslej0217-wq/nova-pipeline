# 외부 커뮤니티 반응 수집

## 목표
다양한 커뮤니티에서 실시간 반응 수집해서
콘텐츠 방향에 반영

## 수집 플랫폼
Tavily로 검색:
- "{주제} 디시인사이드 반응"
- "{주제} 네이버 카페 반응"
- "{주제} 블라인드 반응"
- "{주제} 레딧 reaction"
- "{주제} 트위터 반응"

## 분석 항목
- 가장 많이 언급된 키워드 → 다음 주제 후보
- 부정 반응 → brain/패턴.md 차단 목록
- 긍정 반응 → brain/개인화.md 성공 패턴
- "더 알고 싶다" 반응 → 시리즈 기획 연결
- 논쟁 많은 주제 → 높은 참여율 예측

## 커뮤니티별 특성
- 디시/블라인드: 솔직한 반응, 트렌드 빠름
- 네이버 카페: 30~50대 반응
- 레딧: 글로벌 트렌드 선행 지표
- 트위터: 실시간 바이럴 감지

## 실행 주기
10회마다 자동 실행
결과 memory/커뮤니티반응.md에 저장

## 활용
피드백루프 + 댓글분석과 합쳐서
다음 주 캘린더 주제 선택에 반영
