# 성과 기반 자기학습

## 목표
Supabase에 쌓인 점수 데이터를 분석해서
다음 콘텐츠 전략을 자동으로 개선한다

## 실행 순서
1. Supabase에서 최근 20개 기록 조회
   (date, topic, score_total, score_hook, score_trend)
2. score_total 7점 이상 → 성공 패턴으로 분류
3. score_total 5점 이하 → 실패 패턴으로 분류
4. 성공 패턴에서 공통점 추출
   - 주제 유형
   - 훅 문장 스타일
   - 트렌드 반영도
5. brain/패턴.md 자동 업데이트

## 패턴 업데이트 규칙
- 성공 패턴 3개 이상 쌓이면 → 우선순위 상단으로
- 실패 패턴 2번 이상 반복되면 → 자동 차단 목록 추가
- 매 5회 실행마다 자기학습 1회 실행

## 학습 트리거
- 콘텐츠 생성 5회마다 자동 실행
- 품질점수 70점 이하 즉시 실행
- 준호가 "학습해" 명령 시 즉시 실행

## 학습 데이터 수집 (고도화)
- 최근 20개 품질점수 분석
- TOP 5 공통점 / BOTTOM 5 공통점 추출
- 플랫폼별 성과 차이 분석

## 학습 결과 적용
- 잘된 패턴 → skills/ 해당 파일 자동 반영
- 안된 패턴 → brain/실패패턴.md 추가
- 개선사항 → memory/학습이력.md 기록

## 실패 패턴 블랙리스트 (brain/실패패턴.md 누적)
- 반응 낮았던 제목 패턴
- 참여율 낮았던 콘텐츠 구조
- 클릭 안된 썸네일 패턴
→ 이후 생성 시 자동 회피

## 성장 추적
매월 1일 자동 생성: memory/성장리포트_[년월].md
- 이번달 평균 품질점수 / 전월 대비 성장률
- 가장 많이 개선된 영역 / 다음달 집중 개선 목표
