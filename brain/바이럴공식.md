# 바이럴 공식 뇌

## 목표
조회수 터지는 구조를 자동으로 적용

## 바이럴 공식 5가지
1. 반전 공식: 예상 → 뒤집기
   예: "열심히 하면 안 됩니다"
2. 숫자 공식: 구체적 숫자로 신뢰감
   예: "3일 만에", "87% 줄었어요"
3. 비밀 공식: 나만 아는 정보
   예: "아무도 안 알려주는"
4. 공감 공식: 독자 상황 정확히 묘사
   예: "야근하고 집에 왔을 때"
5. 긴급 공식: 지금 안 하면 손해
   예: "지금 안 하면 늦어요"

## 적용 방법
콘텐츠 생성 시 5가지 중 1개 이상 반드시 적용
훅에 1개, 마무리에 1개 기본

## 성과 추적
어떤 공식이 score_total 높은지 Supabase에서 분석
10회 쌓이면 → 최강 공식 자동 선정
