# 텔레그램 에러 알림 시스템

## 목표
시스템 에러 발생 시 junho 휴대폰으로 즉시 알림 전송

## 알림 전송 코드
```bash
source .env
curl -s "https://api.telegram.org/bot$TELEGRAM_TOKEN/sendMessage" \
  --data-urlencode "chat_id=$TELEGRAM_CHAT_ID" \
  --data-urlencode "text=[메시지 내용]"
```

## 알림 발송 조건
아래 상황에서 무조건 텔레그램 알림:
- 인증 3회 실패 → "ALERT: 인증 실패 3회 - 시스템 잠금됨"
- Supabase 연결 실패 → "ALERT: Supabase 연결 실패"
- API 비용 한도 초과 → "ALERT: API 비용 한도 초과"
- 콘텐츠 생성 연속 3회 실패 → "ALERT: 콘텐츠 생성 실패 3회"
- 폴백 시스템 작동 시 → "INFO: 폴백 모델로 전환됨"
- Make.com 웹훅 실패 → "ALERT: Make.com 웹훅 호출 실패"
- 보안 침해 감지 → "CRITICAL: 보안 침해 감지"

## 알림 레벨
- INFO: 참고용 (정상 범위 내 이벤트)
- ALERT: 확인 필요 (비정상 상황)
- CRITICAL: 즉시 대응 필요 (시스템 위험)

## 주간 정상 보고
