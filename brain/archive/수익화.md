## 진짜 패시브 수입 (수동 작업 0%)

### A. YouTube 무얼굴(Faceless) 채널 자동화
완전 자동 파이프라인:
```
트렌드 수집 (Groq) → 스크립트 생성 (Claude/Groq)
→ 보이스오버 (ElevenLabs API, 월 $5)
→ 영상 조합 (Pictory API or Remotion)
→ YouTube Data API로 자동 업로드
```
- RPM(1000뷰당 수익): $3~15
- 50,000뷰/월 기준: $150~750 (약 20~100만원)
- 설정 후 손댈 것 없음 — 진짜 패시브
- 현재 nova-pipeline과 연결 가능 (brain/플랫폼_youtube.md 참조)

### B. 대역폭 공유 (Bandwidth Sharing)
남는 인터넷 대역폭을 자동 판매:
```bash
# Docker로 실행 (서버/PC 항상 켜져 있을 때)
docker run -d --name honeygain \
  honeygain/honeygain:latest -tou-accept \
  -email YOUR_EMAIL -pass YOUR_PASS -device nova-server
```
- Honeygain: 10GB = $0.1, 월 $20~50 가능
- EarnApp (Bright Data): 비슷한 구조
- Vercel 서버리스에서는 불가 → 로컬 PC/VPS 필요
- 완전 패시브: 설정 후 방치

### C. AI SaaS 워크플로우 판매 (n8n)
n8n 자동화 템플릿을 만들어 마켓에 판매:
- n8n 공식 템플릿 마켓 → 한 번 업로드, 반복 수익
- 가격: 템플릿당 $9~49
- AI 관련 워크플로우 수요 폭증 중 (183k GitHub stars)
- 코드 없이 시각적 플로우 빌더 → 제작 빠름
