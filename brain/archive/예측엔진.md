# 주제 예측 엔진

## 목표
Supabase 성과 데이터 + 트렌드 조합해서
다음 주 어떤 주제가 잘 될지 스스로 예측

## 실행 순서
1. Supabase에서 전체 기록 조회
2. score_total 기준 상위 3개 패턴 추출
3. brain/트렌드.md 읽기
4. 패턴 + 트렌드 교차 분석
5. 다음 주 추천 주제 5개 생성
6. 각 주제별 예상 점수 예측
7. 결과 memory/예측.md에 저장

## 예측 기준
- 과거 7점 이상 주제 유형 → 높은 가중치
- 트렌드와 겹치는 키워드 → 우선 선택
- 최근 3일 이내 다룬 주제 → 제외
- 댓글 유도 문장 있었던 구조 → 재사용
