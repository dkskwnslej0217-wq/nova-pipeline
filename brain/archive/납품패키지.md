# 외부 납품 패키지

## 목표
지금 만든 시스템을 다른 사람/기업에 납품 가능한 구조로 포장

## 납품 유형

### A. 콘텐츠 자동화 패키지
대상: 유튜버, 인스타그래머, 블로거
포함: brain/ + skills/ + run_core.md + run_weekly.md
단가: 50~150만원

### B. 리서치 에이전트 패키지
대상: 컨설팅사, 스타트업
포함: skills/시장조사.md + skills/보고서생성.md + Tavily
단가: 100~300만원

### C. 업무 자동화 패키지
대상: 소상공인, 중소기업
포함: skills/이메일자동화.md + skills/보고서생성.md
단가: 50~200만원

### D. 풀패키지
대상: 기업 전체 도입
포함: 전체 시스템
단가: 500만원~

## 납품 프로세스
1. 고객 요구사항 파악
2. 필요한 skills/ 파일만 선별
3. CLAUDE.md 고객용으로 커스터마이징
4. .env 파일 고객 API 키로 교체
5. 테스트 실행 후 납품
6. 사용법 문서 자동 생성

## 사용법 문서 자동 생성
납품 시 Claude Code가 자동으로 생성:
- 설치 가이드
- API 키 발급 방법
- 실행 방법
- FAQ
