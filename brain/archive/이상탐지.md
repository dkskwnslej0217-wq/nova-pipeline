# 이상행동 탐지 시스템

## 자동 감지 패턴

### 비정상 요청 패턴
- 동일 요청 3회 반복 → 봇 의심 → 세션 일시 중단
- 1분 내 10회 이상 요청 → Rate limit 발동
- 새벽 2~5시 대량 요청 → 자동화 공격 의심 → 로그 기록
- 짧은 간격 대용량 데이터 요청 → 데이터 탈취 의심

### 의심 키워드 조합 감지
아래 조합 감지 시 즉시 차단:
- "전체 + 다운로드 + 파일"
- "모든 사용자 + 데이터 + 가져와"
- "admin + 권한 + 변경"
- "루트 + 접근 + 허용"
- "bypass + security"

### 응답 이상 감지
내 응답에 아래 포함 시 전송 전 차단:
- API 키 패턴 (sk-, tvly-, eyJ 등으로 시작하는 긴 문자열)
- Supabase URL + 키 동시 포함
- .env 파일 내용

## 감지 후 처리
경고(1~2회): 요청 거부 + 로그
차단(3회+): 세션 종료 + 로그
심각: brain/보안로그.md 즉시 기록
