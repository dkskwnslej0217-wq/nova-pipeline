# 트렌드 예보 뇌

## 목표
지금 뜨는 것보다 앞으로 뜰 것을 먼저 잡는다

## 검색 키워드
- "upcoming AI trends 2026"
- "next big thing tech 2026"
- "viral content trend prediction"

## 분석 방법
1. 해외에서 뜨는 것 → 한국 도입 3~6개월 후
2. 기술 뉴스에서 화제 → 일반인 관심 1~3개월 후
3. 미국 유튜브 인기 주제 → 한국 유행 2~4개월 후

## 수집 주기
- 매일 오전 7시: YouTube API(한국 급상승) + Tavily(뉴스) 병렬 수집 → memory/트렌드_[날짜].md 저장
- 매주 월요일: 예보 자동 실행 → memory/주간트렌드예보.md 저장

## 수집 소스 우선순위
1. YouTube 급상승 TOP 10 (YOUTUBE_API_KEY — 직접 수집, 실시간) ← 업그레이드
2. 네이버 실시간 검색 (Tavily) / 3. 구글 트렌드 (Tavily)
4. 미국 YouTube 급상승 (YOUTUBE_API_KEY, regionCode=US) — 선행 트렌드
5. 인스타그램 인기 해시태그 / 6. X 실시간 트렌드

## 트렌드 분석 필터
- 준호 콘텐츠 주제 연관성 70점 이상만 활용
- 일시적 이슈 vs 지속 트렌드 구분
- 민감한 사회적 이슈 자동 제외

## 콘텐츠 생성 시 적용
- 당일 트렌드 TOP 3 자동 반영
- 트렌드 키워드 제목·해시태그에 삽입
- 트렌드 연관 각도로 주제 재해석

## 주간 예보 형식 (memory/주간트렌드예보.md)
- 이번주 예상 인기 주제 TOP 5
- 플랫폼별 추천 콘텐츠 방향
- 지난주 트렌드 적중률

## 활용
예측 엔진과 연동 / 미래 트렌드 주제 캘린더 앞쪽에 배치
