# 개인화 학습

## 목표
내 말투, 스타일, 잘 된 훅 패턴을 자동으로 학습해서
점점 나만의 AI로 진화

## 학습 항목
1. 훅 문장 패턴
   - output/ 폴더에서 score_hook 8점 이상 콘텐츠 분석
   - 공통 구조 추출 (반전형/숫자형/질문형 등)
   - 성공 훅 패턴 목록 자동 업데이트

2. 말투/스타일
   - 짧고 직관적인 문장
   - 구어체 위주
   - 전문용어 최소화

3. 댓글 유도 패턴
   - score_comment 8점 이상 콘텐츠 분석
   - 잘 된 댓글 유도 문장 구조 추출
   - 업데이트 주기: 10회 실행마다

## 적용 방법
콘텐츠 생성 시 이 파일 읽고
성공 패턴 우선 적용

---

## 사용자 프로필 구조 (Supabase users 테이블)
user_id / 이름 / 주력 플랫폼 / 콘텐츠 스타일(전문적·친근한·중립)
선호 주제 TOP3 / 기피 주제 / 최고 성과 콘텐츠 유형
활동 시간대 / 누적 생성 횟수 / 평균 품질점수

## 개인화 적용 규칙
콘텐츠 생성 시:
- 해당 유저 프로필 먼저 로드
- 선호 스타일 자동 조정 / 기피 주제 자동 회피
- 최고 성과 패턴 우선 적용

## 프로필 자동 업데이트
10회 생성마다: 최근 성과로 프로필 갱신 → memory/프로필이력_[유저ID].md 기록

## 멀티유저 격리
- Supabase RLS로 유저별 완전 격리
- 프로필 로드 시 user_id 검증 필수
- 교차 오염 감지 시 즉시 차단 + 로그

## VIP 사용자
누적 100회↑ or 6개월↑ → VIP 태그 자동 부여 → 맞춤 리포트 월 1회 자동 생성
