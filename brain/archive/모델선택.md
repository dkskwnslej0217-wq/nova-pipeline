# 모델 자동 선택

## 목표
작업 복잡도에 따라 무료/유료 모델 자동으로 선택
비용 최소화

## 모델 등급

### 무료 모델 (간단한 작업)
- Gemini Flash (구글) — 빠르고 무료 한도 넉넉
- Groq (Llama3) — 엄청 빠름, 무료 한도 있음
- Claude Haiku — 가장 저렴한 Claude

### 유료 모델 (복잡한 작업)
- Claude Sonnet — 중간 복잡도
- Claude Opus — 최고 복잡도

## 작업별 모델 선택 기준

### 무료 모델로 충분한 것
- 단순 텍스트 요약
- 짧은 콘텐츠 초안 (500자 이내)
- 단순 번역
- 키워드 추출
- 간단한 분류 작업

### Claude Haiku로 충분한 것
- 중간 길이 콘텐츠 생성
- 기본 분석
- 간단한 코드 수정

### Claude Sonnet 필요한 것
- 복잡한 콘텐츠 생성
- 멀티에이전트 파이프라인
- 긴 파일 여러 개 동시 처리

### Claude Opus 필요한 것
- 최고 품질 콘텐츠
- 복잡한 전략 판단
- 중요한 의사결정

## 비용 절감 전략
- 트렌드 수집 → Gemini Flash
- 초안 생성 → Claude Haiku
- 검수/편집 → Claude Sonnet
- 최종 판단 → Claude Sonnet

## API 키 관리
.env 파일에 추가:
GEMINI_API_KEY=여기에_구글_API_키
GROQ_API_KEY=여기에_Groq_API_키

## 무료 API 발급
- Gemini Flash: aistudio.google.com (무료)
- Groq: console.groq.com (무료)
