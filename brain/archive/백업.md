# 자동 백업 시스템

## 목표
중요 데이터 자동 백업해서 복구 가능하게 유지

## 백업 대상
- brain/ 폴더 전체 (판단/학습 규칙)
- memory/ 폴더 전체 (기억/캘린더)
- CLAUDE.md (핵심 설정)
- run_core.md / run_weekly.md

## 백업 방법
output/backup/[날짜]/ 폴더에 자동 저장

## 백업 주기
- 매일 실행 종료 시 자동 백업
- 주요 파일 변경 시 즉시 백업

## 복구 방법
output/backup/ 에서 최신 백업 찾아서 복원

## Supabase 백업
매주 월요일 전체 데이터 export
output/backup/supabase_[날짜].json 저장
