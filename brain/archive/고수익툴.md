## TOP 툴 목록

### 1. n8n — 183k ★ (시각적 워크플로우 자동화)
- GitHub: n8n-io/n8n
- 특징: 400+ 앱 연결, AI 노드 내장 (LangChain 네이티브 지원)
- nova-pipeline 연동법:
  ```
  Webhook 트리거 → HTTP Request → nova-pipeline /api/run-pipeline
  ```
- 무료 사용: Self-hosted (Docker)
  ```bash
  docker run -d --name n8n -p 5678:5678 n8nio/n8n
  ```
- 수익화: n8n 템플릿 마켓에 AI 워크플로우 판매 ($9~49/개)
- 핵심 AI 노드: AI Agent, OpenAI, Anthropic, Groq, Vector Store

### 2. LangChain — 100k+ ★ (AI 에이전트 프레임워크)
- GitHub: langchain-ai/langchain
- 특징: LLM 체인, 에이전트, 메모리, 툴 통합
- nova-pipeline 연동법:
  ```javascript
  // Vercel 함수에서 LangChain 사용
  import { ChatGroq } from "@langchain/groq";
  import { HumanMessage } from "@langchain/core/messages";
  
  const model = new ChatGroq({ model: "llama-3.3-70b-versatile" });
  const response = await model.invoke([new HumanMessage(prompt)]);
  ```
- 핵심 기능: RAG, 에이전트, 멀티스텝 체인
- 비용: Groq 백엔드 사용 시 무료

### 3. Crawlee — 15k+ ★ (웹 스크래핑)
- GitHub: apify/crawlee
- 특징: Puppeteer/Playwright 기반, anti-bot 우회
- nova-pipeline 연동: 경쟁자 콘텐츠 수집, 트렌드 감지

### 4. AgentGPT / AutoGPT 계열
- 자율 에이전트: 목표만 주면 스스로 태스크 분해 + 실행
- nova-pipeline에서: 콘텐츠 리서치 → 생성 → 발행 전 과정 위임 가능

## nova-pipeline 즉시 적용 가능한 조합

### 조합 A: n8n + Groq (트렌드 수집 강화)
```
n8n 스케줄러(08:00 KST)
→ Reddit/HN/GitHub 동시 수집 (HTTP Request 병렬)
→ Groq로 요약
→ nova-pipeline /api/run-pipeline에 POST
```
Make.com 대체 또는 보조로 사용 가능

### 조합 B: LangChain RAG (콘텐츠 품질 향상)
```
업로드된 이전 성공 콘텐츠 → Vector DB (Supabase pgvector)
→ 신규 콘텐츠 생성 시 유사 성공 패턴 참조
→ 품질 일관성 유지
```

### 조합 C: YouTube 자동화 풀스택
```
Groq (스크립트) → ElevenLabs (음성) → Remotion (영상)
→ YouTube Data API (자동 업로드) → Supabase (로그)
```
brain/영상생성.md 참조
