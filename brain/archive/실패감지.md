# 실패 감지 루프

## 목표
낮은 점수 패턴을 감지하고
자동으로 전략을 수정한다

## 실패 기준
- score_total 5점 이하
- 자기검증 3개 이상 실패
- 같은 주제 유형 연속 2회 낮은 점수

## 감지 시 자동 실행
1. Supabase에서 최근 실패 기록 조회
2. 실패 원인 분류
   - 훅 약함 → brain/개인화.md 훅 패턴 교체
   - 트렌드 미반영 → 멀티소스 수집 강화
   - 중복 주제 → 금지 목록 추가
3. brain/패턴.md 자동 업데이트
4. 다음 실행에 수정된 전략 적용
