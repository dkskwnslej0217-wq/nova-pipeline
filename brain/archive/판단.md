# 판단 기준

## 주제 선택 기준
- Supabase에서 최근 10개 기록 조회
- 조회수/반응 좋았던 주제 패턴 파악
- 트렌드.md와 대조해서 겹치는 주제 우선선택
- 같은 주제 3일 이내 반복 금지

## 콘텐츠 품질 기준
- 훅 문장이 있는가?
- 댓글 유도 문장이 있는가?
- 15자 이내 문장으로 구성됐는가?
- 없으면 자동 재작성

## 콘텐츠 카테고리 분리
- 일반 트렌드 콘텐츠: 트렌드 기반 자동 생성
- 코딩 교육 콘텐츠: brain/코딩교육.md 기반 생성
- 납품/비즈니스 콘텐츠: brain/납품패키지.md 기반 생성

## 중복 방지 규칙
- 같은 카테고리 내에서만 중복 체크
- 코딩 교육 주제는 트렌드 주제와 별도 관리
- 캘린더에서 카테고리별로 날짜 배치:
  월수금: 트렌드 콘텐츠
  화목: 코딩 교육 콘텐츠
  토: 납품/비즈니스 콘텐츠

## 자기점수
실행 후 스스로 1~10점 매기기
- 트렌드 반영도
- 훅 강도
- 댓글 유도력
점수 Supabase에 함께 저장
