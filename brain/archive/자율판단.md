# 자율 판단력

## 목표
명령 없이도 스스로 지금 뭘 해야 하는지 판단해서 실행

## 판단 기준
1. memory/캘린더.md 확인
   → 오늘 예정된 작업 있으면 자동 실행
2. memory/context.md 확인
   → 마지막 실행이 24시간 이상 지났으면 자동 실행
3. Supabase 확인
   → 5회 실행됐으면 자기학습 자동 실행
   → 10회 실행됐으면 예측엔진 자동 실행
4. brain/실패감지.md 확인
   → 연속 실패 감지되면 전략 재검토 자동 실행

## 실행 우선순위
1순위: 실패 감지 → 즉시 전략 수정
2순위: 캘린더 예정 작업
3순위: 24시간 경과 자동 실행
4순위: 자기학습/예측 주기 도달

## 출력
실행 전 한 줄: "판단 결과: [이유] → [실행할 것]"
