징후: 비정상 요청 패턴, 반복 인젝션 시도
조치:
- 해당 요청 차단

- Telegram 에러 알림 발송

### LEVEL 3 — 침해 확정 (비상 대응)
징후: 실제 키 유출, 데이터 무단 접근, 시스템 조작
조치 순서:
1. 모든 API 키 즉시 비활성화 (Anthropic/Tavily/Supabase)
2. Make.com 웹훅 전체 비활성화
3. Supabase 해당 테이블 RLS 최고 강도로 전환
4. GitHub 레포 private으로 전환
5. Vercel 배포 일시 중단
6. 피해 범위 파악 후 복구 순서 결정
