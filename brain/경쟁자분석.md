# 경쟁자 분석 시스템

## 분석 대상
memory/경쟁자목록.md에 준호가 직접 등록
- 플랫폼별 주요 경쟁 계정 최대 10개
- 분야: 투자교육 / AI자동화 / 콘텐츠 크리에이터

## 주간 자동 분석 (매주 수요일)
Tavily로 경쟁자 최신 콘텐츠 수집:
- 최근 7일 발행 콘텐츠 수
- 평균 반응률 (좋아요/댓글/공유)
- 인기 콘텐츠 TOP 3 주제
- 사용 해시태그 패턴
- 업로드 시간대

## 분석 결과 활용
- 경쟁자 잘된 주제 → 준호 스타일로 재해석
- 경쟁자 안한 주제 → 블루오션 기회로 기록
- 경쟁자 해시태그 → 겹치지 않는 차별화 전략
- 결과 memory/경쟁자분석_[날짜].md 저장

## 차별화 전략 자동 추출
분석 후 자동 생성:
- 경쟁자 대비 준호만의 강점 3가지
- 경쟁자가 놓친 틈새 주제 3가지
- 이번주 차별화 콘텐츠 방향 제안

## 벤치마크 추적 (분기별)
- 준호 성장률 vs 경쟁자 성장률
- 플랫폼별 포지션 변화
- 시장 내 위치 평가
