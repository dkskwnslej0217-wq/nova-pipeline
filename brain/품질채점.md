# 콘텐츠 품질 자동채점 시스템

## 채점 기준 (100점 만점)

### 제목 (25점)
- 숫자 포함 여부 +5점
- 감정 자극 단어 포함 +5점
- 검색 키워드 포함 +5점
- 15자 이내 간결함 +5점
- 궁금증 유발 구조 +5점

### 본문 구조 (25점)
- 첫 문장 3초 후킹 +10점
- 기승전결 구조 완성 +5점
- 문단 3줄 이내 +5점
- 행동 유도 문구 포함 +5점

### 플랫폼 최적화 (25점)
- 해시태그 적정 개수 +10점
- 플랫폼별 권장 길이 준수 +10점
- 이모지 적절 사용 +5점

### 차별화 (25점)
- 트렌드 키워드 반영 +10점
- 독창적 각도 +10점
- 중복 콘텐츠 아님 +5점

## 점수별 처리
- 90점↑: 즉시 발행
- 70~89점: 약점 항목 자동 보완 후 재채점
- 70점↓: 전면 재생성
- 재생성 후도 70점↓: 준호에게 수동 검토 요청

## 채점 결과 저장 (Supabase content 테이블)
콘텐츠 ID / 항목별 점수 / 총점 / 보완 내용 / 최종 발행 여부
