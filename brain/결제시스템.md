# 결제 시스템

## 단계별 결제 수단

### 1단계 (지금 ~ 100명) — 수동
- 카카오뱅크 계좌이체
- 입금 확인 후 Supabase에서 수동으로 plan_type 변경:
  update users set plan_type='starter', paid_until='2026-04-29'
  where user_id='유저명';

### 2단계 (100명 이후) — 반자동
- 토스페이먼츠 or 페이플 연동
- 결제 완료 → Make.com 웹훅 → Supabase 자동 업데이트

### 3단계 (사업자 등록 후) — 완전 자동
- 아임포트(포트원) API 연동
- 카카오페이 + 토스 + 카드 전부 연동
- 결제 → Supabase 자동 업데이트 → 이메일 발송

## 가격표
- 무료: 하루 5회
- 스타터 (월 1만원): 하루 20회
- 프로 (월 3만원): 무제한
- 팀 (월 10만원): 멀티유저 5명

## 지금 당장 할 수 있는 것
카카오뱅크 계좌번호 → 콘텐츠 끝에 삽입
"더 많이 쓰고 싶으면 → 스타터 플랜 월 1만원"

## Make.com 결제 자동화 시나리오
트리거: 토스페이먼츠 결제 완료 웹훅
액션 1: Supabase users 테이블 plan_type 업데이트
액션 2: 구글시트에 결제 기록 저장
액션 3: 환영 이메일 자동 발송
