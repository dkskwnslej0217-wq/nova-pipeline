# 시리즈 기획 뇌

## 목표
1편~10편 시리즈 자동 설계
구독자가 다음 편 기다리는 구조 만들기

## 시리즈 구조 공식
1편: 문제 제기 (왜 이게 중요한가)
2편: 핵심 개념 (기초 설명)
3편: 실전 방법 1 (쉬운 것)
4편: 실전 방법 2 (심화)
5편: 실수 모음 (공감 유도)
6편: 도구 소개 (실용 정보)
7편: 비교 분석 (논쟁 유도)
8편: 성공 사례 (희망 유도)
9편: 심화 팁 (충성 구독자용)
10편: 총정리 (신규 유입용)

## 시리즈 생성 방법
주제 입력 시 "시리즈로 만들어줘" 명령 가능
→ 10편 제목 + 핵심 내용 자동 생성
→ memory/시리즈.md에 저장
→ 캘린더에 자동 배치

## 클리프행어 규칙
매 편 마지막에 다음 편 예고
"다음 편에서는 [구체적 내용] 알려드릴게요"

## 시리즈 감지 트리거
- 단일 주제가 콘텐츠 3개 이상 필요한 경우
- "시리즈로 만들어줘" 명령 시
- 인기 콘텐츠 후속편 요청 감지 시

## 시리즈 자동 설계 (감지 후)
1. 전체 회차 수 결정 (3~10회)
2. 회차별 주제 목차 작성
3. 각 회차 예상 키워드 선정
4. 발행 간격 추천 (매일/격일/주 1회)
5. 시리즈 전체 해시태그 세트 설계

## 시리즈 관리 (memory/시리즈목록.md 자동 등록)
시리즈명 / 총 회차 / 완료 회차 / 다음 발행 예정일 / 평균 성과

## 시리즈 완결 처리
- 전체 성과 분석 → 최고 반응 회차 패턴 추출
- brain/승리패턴.md 시리즈 공식 추가
- 후속 시리즈 기획 자동 제안

## 시리즈 중단 감지
3회↑ 발행 후 성과 급락 시 → 중단 또는 방향 전환 제안 + 원인 분석
