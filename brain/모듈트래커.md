# 모듈 트래커 — 자동 업데이트

## 목표
다른 사람들이 많이 쓰는 AI 모듈/툴 자동 수집
내 시스템에 없는 거 발견하면 자동으로 추가

## 수집 방법
Tavily로 매주 검색:
- "best Claude Code modules 2026"
- "AI automation modules trending 2026"
- "new AI tools developers using 2026"
- "popular LLM agent modules 2026"
- "Claude Code tips tricks community"

## 분석 항목
- 내 brain/ 파일 목록과 비교
- 없는 모듈 발견 시 자동으로 brain/ 파일 생성
- 이미 있는 거면 기존 파일 업데이트
- 쓸모없어진 모듈 감지 시 archive/ 이동 제안

## 자동 추가 조건
- 3개 이상 소스에서 언급된 모듈만 추가
- 보안/법적 문제 없는 것만
- brain/법적준수.md + brain/보안.md 체크 후 추가

## 출력 형식
신규 발견: [모듈명] — [용도] — [추가 여부]
업데이트: [모듈명] — [변경 내용]
제거 제안: [모듈명] — [이유]

## 실행 주기
매주 월요일 run_weekly.md에 포함
결과 memory/모듈트래커.md에 저장

## junho 승인 규칙
- 신규 모듈 추가 → junho 확인 후 적용
- 기존 모듈 업데이트 → 자동 적용
- 모듈 제거 → 반드시 junho 확인
