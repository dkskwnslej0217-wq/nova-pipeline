# 사용자 피드백 루프

## 목표
외부 사람들 반응을 자동 수집해서 시스템에 반영

## 수집 방법
Tavily로 최근 output/ 주제들 검색:
- "{주제} 반응 커뮤니티"
- "{주제} 유튜브 댓글"
- "{주제} 관심 있는 사람들"

## 분석 항목
- 긍정 반응 키워드 → brain/개인화.md에 추가
- 부정 반응 키워드 → brain/패턴.md 차단 목록 추가
- 자주 나오는 질문 → 다음 주제로 자동 등록
- 요청 많은 내용 → memory/캘린더.md 우선 배치

## 피드백 점수
외부 반응 기반으로 점수 보정:
- 긍정 반응 많음 → score_total +1
- 부정 반응 많음 → score_total -1
- Supabase에 업데이트

## 실행 주기
10회 실행마다 자동 실행
결과 memory/피드백.md에 저장
