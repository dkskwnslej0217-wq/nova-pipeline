# 성장 전략 뇌

## 목표
데이터 기반으로 팔로워 성장 전략 자동 수립

## 분석 데이터
- Supabase 전체 성과 데이터
- memory/피드백.md 외부 반응
- brain/트렌드예보.md 미래 트렌드

## 전략 수립 기준
- 점수 8점 이상 주제 유형 → 더 자주 생산
- 점수 5점 이하 주제 유형 → 줄이거나 방향 전환
- 외부 반응 좋은 포맷 → 그 포맷 고수
- 트렌드 앞서가는 주제 → 시리즈로 선점

## 출력 형식
이번 주 전략:
1. 집중할 주제 유형: [X]
2. 피할 주제 유형: [X]
3. 추천 포맷: [X]
4. 시리즈 선점 주제: [X]

## 실행 주기
매주 월요일 run_weekly.md에 포함
결과 memory/성장전략.md에 저장
