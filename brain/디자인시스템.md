# 디자인 시스템

## 핵심 원칙
- 매주 월요일 자동 업데이트
- 데이터 기반 트렌드 반영
- 핵심 브랜드는 유지, 나머지는 트렌드 따라감

## 자동 업데이트 구조
brain/디자인트렌드.md → 이번 주 트렌드 수집
→ 사람들이 많이 쓰는 디자인 패턴 분석
→ brain/웹자동업데이트.md → 웹사이트 자동 반영

## 데이터 수집 방법
Tavily로 매주 검색:
- "most popular website design 2026"
- "trending UI design patterns 2026"
- "top landing page designs this week"
- "viral app design trends"
- "한국 인기 웹디자인 트렌드"

## 분석 항목
- 이번 주 가장 많이 쓰이는 색상
- 가장 인기 있는 레이아웃 구조
- 트렌드 폰트
- 인기 애니메이션 효과
- 사람들이 오래 머무는 디자인 패턴

## 변경 주기
매주 월요일:
- 배경 색상 → 트렌드 색상으로 교체
- 폰트 → 트렌드 폰트로 교체
- 레이아웃 → 트렌드 구조 반영
- 애니메이션 → 최신 효과 적용

## 핵심 브랜드 (절대 변경 금지)
- 로고: 미정 (나중에 직접 입력)
- 브랜드명: 미정
- 핵심 메시지: 미정

## 변경 심각도
- 낮음: 색상/폰트 미세 조정 → 자동 적용
- 중간: 레이아웃 변경 → junho 확인 후 적용
- 높음: 전체 개편 → junho가 직접 결정
