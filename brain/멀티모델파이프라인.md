# 멀티모델 파이프라인

## 분업 구조
```
0단계 YouTube Data API → 한국 급상승 TOP10 실데이터 수집
1단계 Gemini Flash     → 트렌드 요약 + 키워드 추출 (실패→Groq폴백)
2단계 Groq 70b        → 훅 초안 3가지 생성
3단계 Claude          → 품질검수 + 브랜드톤 + 최종완성
```
폴백: Gemini429→Groq대체 / Groq실패→Claude단독 / 전체실패→에러알림.md→텔레그램

## 실패 조건
| 모델 | 실패 조건 | timeout |
|------|-----------|---------|
| Gemini | 429(할당초과) / 5xx | 10s |
| Groq | 429 / 5xx | 8s |
| Claude | 5xx | 15s → 전체실패 시 에러알림.md |
