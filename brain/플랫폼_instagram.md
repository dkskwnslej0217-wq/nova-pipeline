# Instagram 플랫폼 규정

## 콘텐츠 타입
- 이미지 필수 (텍스트만 불가)
- Pollinations.ai 자동 이미지 생성 사용 중
- 이미지 비율: 1:1 (1080x1080)

## 글자 수 제한
- 캡션: 최대 2,200자
- 해시태그: 최대 30개 (3~5개 권장)

## 하루 발행 한도
- 최대 25개 (초과 시 계정 제한 위험)
- 현재 파이프라인: 하루 1회

## 차단 방지 규칙
- 같은 해시태그 반복 사용 금지 (매일 다르게)
- 스팸성 반복 게시 금지
- 짧은 시간 내 대량 게시 금지
- 외부 링크 캡션에 넣으면 도달률 감소

## API 정보
- 엔드포인트: graph.instagram.com/v21.0
- 환경변수: INSTAGRAM_ACCESS_TOKEN, INSTAGRAM_BUSINESS_ACCOUNT_ID
- 토큰 만료: 있음 (주기적 갱신 필요)
- 2단계: 컨테이너 생성 → media_publish

## 콘텐츠 특성
- 비주얼 중심
- 해시태그 효과 큼
- 릴스(영상) 도달률 일반 게시물보다 높음
