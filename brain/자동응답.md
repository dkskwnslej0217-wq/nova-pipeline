# 자동 DM + 댓글 응답

## 목표
팔로워 댓글/DM에 자동으로 맞춤 응답 생성

## 응답 유형

### 댓글 응답
- 질문형 댓글 → 친절한 답변 생성
- 칭찬 댓글 → 감사 + 다음 콘텐츠 예고
- 부정 댓글 → 공감 + 개선 약속

### DM 응답 템플릿
- 구매 문의 → 상품 안내 + 링크
- 협업 문의 → 감사 + 연락처 안내
- 일반 질문 → 친절한 답변

## 응답 규칙
- 항상 친근한 말투
- 이모지 1~2개 포함
- 50자 이내로 간결하게
- 절대 복붙 티 나지 않게
- 댓글마다 약간씩 다르게

## 실행 방법
"자동응답 실행해줘 — [플랫폼] [댓글내용]"
→ 맞춤 응답 3가지 생성
→ 그 중 1개 선택해서 발송

## Make.com 연동 (나중에)
플랫폼 API 연결 후:
→ 댓글 자동 수집 → 응답 생성 → 자동 발송
