# 코딩 교육 커리큘럼

## 목표
코딩 모르는 사람도 Claude Code로
실제 돈 되는 시스템 만들 수 있게 가르치기

## 1단계 — Claude Code 기초 (1주)
- Claude Code 설치 + 설정
- CLAUDE.md 작성법
- 첫 콘텐츠 생성
- 결과물: 첫 자동화 성공

## 2단계 — API 연동 (2주)
- Tavily 웹서치 연동
- Supabase 메모리 연동
- .env 파일 관리
- 결과물: 실시간 트렌드 기반 콘텐츠

## 3단계 — 자동화 (3주)
- Make.com 기초
- 구글시트 연동
- 자동 저장 구조
- 결과물: 완전 자동화 파이프라인

## 4단계 — 수익화 (4주)
- 플랜 구조 설계
- 결제 시스템 연동
- 납품 패키지 만들기
- 결과물: 실제 돈 버는 구조

## 5단계 — 고도화 (2개월)
- 멀티에이전트 파이프라인
- 자기학습 시스템
- 예측 엔진
- 결과물: 스스로 진화하는 AI

## 수익화
- 클래스101 강의: 5만원 × 200명 = 1000만원
- 자체 웹사이트 구독: 월 3만원
- 1:1 코칭: 시간당 10만원
