# 폴백 시스템 (한도 초과 시 자동 전환)

## 목표
claude.ai Pro 한도 가득 차도
사람들이 계속 이용할 수 있게 자동 전환

## 폴백 순서
1순위: claude.ai Pro (지금 쓰는 것)
2순위: Claude API (claude-haiku-4-5 — 저렴)
3순위: Groq API (무료 — 이미 연결됨)
4순위: "잠시 후 다시 시도해주세요" 안내

## 전환 조건
- claude.ai 응답 없음 → 즉시 2순위로
- API 오류 발생 → 다음 순위로
- 모든 한도 초과 → 4순위 (대기 안내)

## Claude API 비용 관리
- claude-haiku 사용 (제일 저렴)
- 입력 토큰 최소화 (토큰최적화.md 적용)
- 월 예산 설정: 10달러 초과 시 Groq로 전환

## .env에 추가 필요
ANTHROPIC_API_KEY=여기에_Claude_API_키
MONTHLY_BUDGET_USD=10

## 자동 전환 로직
run_core.md 실행 시:
1. 현재 사용 중인 모델 확인
2. 응답 실패 시 다음 모델로 자동 전환
3. 전환 이력 memory/폴백로그.md에 기록
4. junho한테 "현재 [모델] 사용 중" 보고

## Anthropic API 키 발급
platform.anthropic.com 접속
→ API Keys → Create Key
→ .env에 ANTHROPIC_API_KEY= 저장
