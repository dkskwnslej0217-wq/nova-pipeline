# 수익화 자동화

## 목표
콘텐츠 → 구독자 → 수익으로 자동 연결되는 구조

## 수익 파이프라인

### 1단계 — 콘텐츠로 신뢰 쌓기 (지금)
- run_core.md로 매일 콘텐츠 자동 생성
- 플랫폼별 최적화 포맷으로 업로드
- 목표: 팔로워 1000명

### 2단계 — 디지털 상품 출시 (팔로워 1000명)
자동 생성할 상품 목록:
- "Claude Code 완전 가이드" PDF
- "AI 자동화 시스템 파일 패키지"
- 가격: 1~5만원
- 판매처: 크몽, 클래스101, 자체 페이지

### 3단계 — 구독 커뮤니티 (팔로워 3000명)
- 월 1~3만원 구독
- 100명 = 월 100~300만원 자동 수익
- 플랫폼: 카카오뱅크 → 페이플

### 4단계 — B2B 납품 (레퍼런스 쌓인 후)
- brain/납품패키지.md 활용
- 건당 50~300만원

## 자동 트리거
팔로워 수 체크 (Tavily로 검색):
- 1000명 돌파 → "디지털 상품 출시 준비" 알림
- 3000명 돌파 → "구독 커뮤니티 오픈 준비" 알림
- 1만명 돌파 → "B2B 납품 본격화" 알림

## 콘텐츠 → 수익 연결 공식
- 콘텐츠 끝에 항상 → 디지털 상품 언급
- 댓글 유도 → DM 유도 → 구매 전환
- 시리즈 콘텐츠 → 마지막 편에 유료 전환
