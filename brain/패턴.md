# 잘 먹히는 글쓰기 패턴 라이브러리

## 훅 패턴 (첫 줄)

### 반전형
"열심히 했는데 왜 안 될까."
"더 노력하면 된다는 말, 이제 믿지 않는다."

### 공감형
"나만 이런 거 아니었구나."
"다들 괜찮은 척하는 거 알아."

### 숫자형
"3가지만 바꿨더니 달라졌다."
"1년 동안 매일 했더니 생긴 일."

### 질문형
"너는 왜 그렇게 사냐는 말, 들어본 적 있어?"
"지금 하는 일이 진짜 하고 싶은 건지 모르겠을 때."

## 본문 구조 패턴

### 문제-원인-해결
훅 → 공감되는 문제 제기 → 진짜 원인 → 해결/팁 → 행동 유도

### 스토리형
훅 → 나의 경험 → 깨달은 것 → 네 얘기이기도 함 → 질문

### 리스트형
훅 → 번호 리스트 (3~5개) → 한 줄 요약 → 행동 유도

- "A야 B야? 솔직하게."
- "이거 공감되면 저장해둬. 나중에 필요함."
