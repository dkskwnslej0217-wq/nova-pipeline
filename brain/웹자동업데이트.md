# 웹사이트 자동 업데이트 시스템

## 목표
매주 디자인 트렌드 수집 → 웹사이트 자동으로 반영

## 작동 구조
1. brain/디자인트렌드.md → 이번 주 트렌드 수집
2. memory/디자인트렌드.md → 트렌드 기록 확인
3. 변경 사항 있으면 → 웹사이트 코드 자동 수정
4. Vercel 자동 배포 (나중에 연결)

## 업데이트 항목
- 색상 팔레트 → 트렌드 색상으로 교체
- 폰트 → 트렌드 폰트로 교체
- 레이아웃 → 트렌드 구조 반영
- 애니메이션 → 최신 효과 적용
- 컴포넌트 → 인기 스타일 반영

## 업데이트 규칙
- 핵심 브랜드 색상은 유지 (변경 금지)
- 폰트는 가독성 우선
- 모바일 최적화 항상 유지
- 변경 전 memory/디자인트렌드.md 백업

## 변경 심각도
- 낮음: 색상/폰트 미세 조정 → 자동 적용
- 중간: 레이아웃 변경 → 사람에게 확인 후 적용
- 높음: 전체 디자인 개편 → 사람이 직접 결정

## 저장
변경 이력 memory/웹업데이트로그.md에 저장
