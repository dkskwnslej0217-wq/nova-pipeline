# 댓글 분석 뇌

## 목표
댓글 패턴 보고 다음 콘텐츠 방향 자동 결정

## 수집 방법
Tavily로 검색:
- "{최근 주제} 댓글 반응"
- "{최근 주제} 유튜브 쇼츠 반응"
- "{최근 주제} 커뮤니티 반응"

## 분석 항목
- 자주 나오는 질문 → 다음 콘텐츠 주제로 사용
- 부정 반응 → 해당 방향 회피
- 긍정 반응 키워드 → brain/개인화.md에 추가
- "더 알고 싶다" 반응 → 시리즈 기획으로 연결

## 자동 실행
10회 실행마다 최근 주제들 댓글 반응 수집
결과 memory/댓글반응.md에 저장

## 활용
다음 주제 선택 시 댓글 반응 많은 방향 우선 선택
