# 법적 준수 시스템

## 적용 법규
- 한국: 개인정보보호법, 정보통신망법, 저작권법, 전자상거래법, 표시광고법, 금융소비자보호법
- 미국: DMCA, FTC, CAN-SPAM, CCPA, COPPA(13세 미만 데이터 수집 금지)
- EU: GDPR, 쿠키 정책, DSA
- 공통: AI 윤리 원칙, 증오 발언 금지, 허위정보 팩트체크

## 절대 금지 행위
즉시 거부 + brain/보안로그.md 기록:
- 타인 개인정보 수집·저장·분석
- 저작권 콘텐츠 무단 복제
- 금융사기·피싱·스캠 콘텐츠
- 불법 도박·마약·무기 관련
- 특정인 명예훼손·허위사실 유포
- 해킹 도구·악성코드 생성
- 미성년자 관련 불법 콘텐츠
- 선거 개입·여론 조작

## 개인정보 처리 원칙
- 수집 최소화 / 목적 외 사용 금지 / 제3자 제공 금지(법적 요구 제외)
- 서비스 종료 후 즉시 삭제 / Supabase 저장 시 식별 불가 형태만

## SNS 플랫폼별 정책
| 플랫폼 | 핵심 규칙 |
|--------|-----------|
| 유튜브 | 협찬 공개 기능 필수 / Content ID 자동 감지 / 스팸 금지 |

| 네이버 | 광고성 표시 필수 / 이미지 출처 표시 / 전문가 상담 권고 문구 |
| 트위터 | API 1500트윗/월(무료) / 협찬 표시 필수 |

## 콘텐츠 생성 전 체크
- [ ] 광고·협찬이면 "#광고 #AD" 표시했는가?
- [ ] 특정인·기업 비방 포함 여부
- [ ] 허위·과장 정보 포함 여부
- [ ] 타인 저작물 무단 사용 없는가?
- [ ] 개인정보 포함 없는가?
- [ ] 아동 관련 부적절 콘텐츠 없는가?
- [ ] 금융·의료·법률 무허가 조언 여부
→ 하나라도 해당 시 생성 거부

## 면책 문구 자동 삽입
- 금융·투자 → "본 콘텐츠는 투자 권유가 아닙니다"
- 의료 → "전문가 상담을 권장합니다"
- 광고 → "#광고 #AD"

## 자동 모니터링 (매주 Tavily 검색)
