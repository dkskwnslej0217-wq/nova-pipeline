## Make.com 시나리오 현황
- 시나리오 1 (MAKE_SHEETS_WEBHOOK): /api/log 웹훅 → type으로 4분기
  - content: Supabase insert + 구글시트 "콘텐츠기록"
  - subscriber: Supabase upsert + 구글시트 "구독자목록"
  - payment: Supabase update + 구글시트 "결제기록" + Gmail 알림
  - error: 구글시트 "에러로그" + Telegram
  - ⚠️ Telegram/Supabase 직접 처리 부분은 제거하고 구글시트만 남길 것
- 시나리오 2: 업로드 자동화 (설계만 완료, 미구현)
