# 자동 저장 시스템

## 목표
세션 끝나도 뭐 했는지 기억하게
모든 작업을 자동으로 기록

## 저장 규칙
아래 상황에서 무조건 자동 저장:
- 파일 생성/수정 시 → memory/변경로그.md에 기록
- 콘텐츠 생성 완료 시 → memory/context.md 업데이트
- 에러 발생/해결 시 → memory/에러로그.md 기록
- 새 모듈 추가 시 → START_HERE.md 자동 업데이트
- 설정 변경 시 → memory/설정로그.md 기록

## 저장 형식
[날짜] [시간] [작업내용] [결과]
예: 2026-03-29 14:30 brain/감사관.md 생성 완료

## 세션 시작 시 자동 복구
새 세션 열리면:
1. memory/오늘요약.md 읽기 (6줄) → 전체 컨텍스트 파악
2. "어제 [X] 했고 오늘은 [Y] 예정" 자동 보고
※ 추가 필요 시에만: 변경로그.md(파일변경이력) / context.md(콘텐츠이력) / START_HERE.md(단계현황)
※ 구버전(변경로그+context+START_HERE 항상 읽기)은 토큰 낭비로 폐기

## START_HERE.md 자동 업데이트 규칙
새 단계 완료할 때마다:
→ START_HERE.md 완료 단계에 ✅ 자동 추가
→ 다음 단계 업데이트
→ 최근 테스트 결과 업데이트
