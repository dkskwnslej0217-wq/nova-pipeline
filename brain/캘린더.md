# 자동 콘텐츠 캘린더

## 목표
예측엔진 + 성과 데이터 기반으로
일주일치 콘텐츠 계획을 자동으로 짠다

## 실행 조건
- 매주 월요일 첫 실행 시 자동 실행
- 또는 memory/캘린더.md 비어있을 때

## 캘린더 구성 규칙
- 하루 1~2개 콘텐츠
- 플랫폼 번갈아가며 (유튜브/인스타/스레드)
- 같은 주제 연속 금지
- score 높았던 주제 유형 우선 배치
- 트렌드 반영도 높은 주제 앞쪽 배치

## 출력 형식
월: [주제] — [플랫폼] — 예상점수 [X]점
화: [주제] — [플랫폼] — 예상점수 [X]점
수: [주제] — [플랫폼] — 예상점수 [X]점
목: [주제] — [플랫폼] — 예상점수 [X]점
금: [주제] — [플랫폼] — 예상점수 [X]점
토: [주제] — [플랫폼] — 예상점수 [X]점
일: 휴식 또는 성과 분석
