# Facebook 플랫폼 규정

## 콘텐츠 타입
- 텍스트만 가능 (현재 설정)
- 이미지/링크 추가 가능하나 현재 미사용

## 글자 수 제한
- 게시글: 최대 63,206자 (현재 코드 반영됨)
- 권장: 40~80자 (짧을수록 도달률 높음)

## 하루 발행 한도
- 명시적 한도 없으나 하루 5회 이상 시 도달률 급감
- 현재 파이프라인: 하루 1회

## 차단 방지 규칙
- 스팸성 반복 게시 금지
- 외부 링크 본문에 넣으면 도달률 감소 (댓글에 달 것)
- 페이지 토큰 만료 시 즉시 갱신 필요

## API 정보
- 엔드포인트: graph.facebook.com/v25.0
- 환경변수: FACEBOOK_ACCESS_TOKEN, FACEBOOK_PAGE_ID
- 페이지 ID: 1023919464144709 (고정)
- 비즈니스 포트폴리오 ID: 801260493054489
- 토큰 만료: 약 60일 (FACEBOOK_TOKEN_EXPIRES_AT 등록됨)

## 토큰 갱신 방법
