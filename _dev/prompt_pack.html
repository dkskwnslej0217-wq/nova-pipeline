PROMPT 01

Full Review Script (Hook → CTA)

Use case: Complete YouTube video narration script

Hook (10s): Start with a relatable pain point or surprising fact about [TOOL NAME] 2. Introduction (15s): What is [TOOL NAME] and what problem does it solve? 3. Core Features (20s): Explain 3 key features with concrete use cases 4. Pros (15s): 2 specific advantages with real examples 5. Cons (10s): 1–2 honest limitations 6.
