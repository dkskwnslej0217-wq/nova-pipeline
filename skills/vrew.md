# Vrew 영상 제작 스킬

## 목표
Claude Code가 스크립트를 생성할 때
Vrew에서 바로 쓸 수 있는 설정값을 함께 출력한다.

---

## 출력 형식 (매번 이 형식으로 출력)

### 📄 스크립트
[내용]

### 🎬 Vrew 설정값

**배경 스타일**
- 유형: 영상 클립 (Vrew 내장 스톡)
- 키워드: [주제 관련 한국어 키워드 3개]
- 분위기: [밝음 / 차분함 / 강렬함 중 선택]

**AI 목소리**
- 언어: 한국어
- 목소리: 지훈 (남성, 차분하고 신뢰감 있는 톤)
- 속도: 1.0x
- 피치: 기본값

**자막 스타일**
- 폰트: 나눔고딕 Bold
- 크기: 48px
- 위치: 하단 중앙
- 색상: 흰색
- 배경: 반투명 검정 (투명도 60%)
- 강조 단어: 노란색 (#FFE135)

**영상 형식**
- 비율: 9:16 (쇼츠/릴스용)
- 해상도: 1080 x 1920
- 길이 목표: 45~60초

---

## 플랫폼별 설정 차이

### YouTube Shorts / Instagram Reels
- 비율: 9:16
- 길이: 45~60초
- 자막: 크게 (48px 이상)
- 첫 3초: 훅 문장만 화면에 크게

### YouTube 일반
- 비율: 16:9
- 길이: 3~8분
- 자막: 중간 크기 (36px)
- 인트로: 15초 이내

### Threads (텍스트만 필요, Vrew 불필요)
- 스크립트만 output/threads/ 에 저장
- Vrew 설정값 출력 생략

---

## 스크립트 작성 규칙

- 1문장 = 1자막 (Vrew가 자동 분리)
- 문장 길이: 15자 이내 권장
- 숫자/강조 단어는 따로 줄바꿈
- 말하듯이 작성 (읽는 글 X)
- 한 문장 끝에 마침표 붙이기

## 나쁜 예
"안녕하세요 오늘은 Claude 자동화 시스템에 대해 알아보겠습니다."

## 좋은 예
"안녕하세요.
오늘 주제는 Claude 자동화예요.
코딩 몰라도 됩니다.
저도 몰랐거든요."

---

## output 저장 규칙

- output/youtube/ → 유튜브 스크립트 + Vrew 설정값
- output/instagram/ → 인스타 스크립트 + Vrew 설정값
- output/threads/ → Threads 텍스트만

파일명: YYMMDD_주제키워드.md
예: 260328_Claude자동화소개.md

---

## 연속 실행 규칙

한 번 실행 후:
1. output/ 저장 완료
2. memory/context.md 업데이트
3. 자동으로 다음 주제 1개 제안
4. "다음 주제로 바로 실행할까요?" 질문
5. 승인 시 즉시 다음 차수 실행
