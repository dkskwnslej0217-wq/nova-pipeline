\# 트렌드 체크 실행 파일



\## 실행 방법

이 파일을 읽으면 아래를 자동으로 실행해줘.



\## 입력받을 것

\- 오늘 날짜

\- 내 채널 주제 카테고리 (예: 라이프스타일 / 음식 / 여행 등)



\## 자동 실행 절차



\### Step 1: 트렌드 수집

skills/트렌드수집.md 스킬 읽고

오늘 날짜 기준 주제 후보 5개 뽑아줘.



\### Step 2: memory/context.md 대조

이미 다룬 주제와 겹치는 후보 제거.

겹치면 다음 순위 후보로 대체.



\### Step 3: 최종 추천

주제 후보 5개 출력 + 오늘 1순위 추천.
